# Suggested commands
- Install deps: `flutter pub get`
- Format Dart files: `dart format <paths>`
- Analyze files: `flutter analyze <paths>`
- Run unit/widget tests: `flutter test`
- Run integration tests: `flutter test integration_test`
- Run on the user's iPhone: `flutter run -d 00008120-00080C362162601E`
- Enter Nix dev environment: `nix develop`
- Inspect repo status: `git status --short`
- Search code fast: `rg <pattern>`
